
# Scrapper tool

Scrape the url


## Installation

Install scrapper with 

```bash
  git clone https://github.com/kunalrawat425/scrapper.git
  cd scrapper
```

```bash
  pip install -r requirements.txt

  uvicorn app.main:app --reload

```
